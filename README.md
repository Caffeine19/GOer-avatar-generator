# GOer-avatar-genrator

用于生成以 GO 语言吉祥物（一只金花鼠）为主题的头像

## 🚀 来自于

“年轻人的第一个 GO 项目”。本人在学习入门了 GO 之后，一直想着写点什么，于是有了这一个一行 GO 代码也没有的项目。
